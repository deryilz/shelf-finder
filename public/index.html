Dublin Shelf Finder
The Dublin Shelf Finder is a tool that helps students find books on the library shelves. To use it, navigate to the Destiny Discover page for a book you want to locate. Then just click the "Find book!" button for your book.
The Shelf Finder was created by Derin Eryilmaz and Rudy Pandit, two 8th graders from Karrer Middle School who noticed the need for an easy and convenient way to locate books. After weeks of work, they released version 1.0.0 on September 17th, 2021.
In late 2025, the tool was rewritten to add more features:
  • Support is no longer limited to one school.
  • Librarians can now log into an admin page and manage shelves.
  • Rotated shelves are supported, as well as shelves of different sizes.
  • The map is more interactive (supports panning and zooming).
This project has been featured on Karrer's WKMS announcements and an official interview with the Dublin City Schools YouTube channel.
Installation
Just install the Dublin Shelf Finder extension! If your school is supported, buttons will be added to Destiny that allow you to open the map.
Contact
This project is open source and has a GitHub repository. There, you can read the source code and report bugs. You can also submit your suggestions for new features, which would be highly appreciated.
If you're a student who wants to help out, or a librarian who's interested in setting up the tool, feel free to email derin@shelf-finder.com! I'm happy to answer any questions.
